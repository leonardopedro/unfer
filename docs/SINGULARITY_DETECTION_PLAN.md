# Implementation Plan: ODEâ†’Hamiltonian Self-Adjointness & Singularity Detection via Hashimoto SIRK

## Goal

Add a self-contained pipeline (`ode_sirk`) that transforms **multivariate polynomial ODEs** into self-adjoint quantum Hamiltonians, detects **essential self-adjointness** (ESA) via Nelson's classical flow-completeness, and localizes **finite-time singularities**. For singular ODEs, it applies coordinate transformations to "straighten" the flow, yielding bounded-time unitary evolutions solvable by the Hashimoto SIRK algorithm.

## 1. Mathematical Specification

### 1.1 ODE â†’ Hamiltonian (Koopman-Weyl Quantization)

Given a polynomial autonomous ODE system in variables $x = (x_1,\dots,x_n)$:
$$ \frac{dx_i}{dt} = f_i(x) , \quad f_i \in \mathbb{R}[x_1,\dots,x_n] $$

Construct the **Weyl-symmetrized Hamiltonian**:
$$ H = \frac{1}{2} \sum_i \left( f_i(x) p_i + p_i f_i(x) \right) = \sum_i \left( f_i(x) p_i - \frac{i}{2} \frac{\partial f_i(x)}{\partial x_i} \right) $$

**Bosonic Mapping (Strict):** To maintain the canonical commutator $[x_i, p_j] = i\delta_{ij}$ required for exact Weyl symmetrization cancellation:
*   $x_i = \frac{1}{\sqrt{2}}(a_i + a_i^\dagger)$
*   $p_i = \frac{-i}{\sqrt{2}}(a_i - a_i^\dagger)$

### 1.2 Essential Self-Adjointness (Nelson's Theorem)

For $D = i(v \cdot \nabla + \frac{1}{2} \text{div } v)$, Nelson's flow-completeness theorem dictates that $D$ is essentially self-adjoint on $C_c^\infty(\mathbb{R}^n)$ **if and only if** the classical flow generated by $v$ is complete (trajectories do not reach infinity in finite time).
*   **Incomplete flow $\implies$ Not ESA:** Time evolution leaks probability.
*   **Complete flow $\implies$ ESA:** SIRK unitary evolution is uniquely defined.

### 1.3 Resolving Singularities via Change of Variables

For a singular flow (e.g., $x' = x^2$), define a diffeomorphism $w = \varphi(x)$ (e.g., $w = 1/x$). 
1. The classical flow transforms to $dw/dt = g(w)$.
2. We construct a *new* Hamiltonian $H_{new}$ from $g(w)$.
3. **Crucial:** Observables must map back. Measuring $x(t)$ requires computing the expectation value $\langle \psi(t) | \varphi^{-1}(\hat{w}) | \psi(t) \rangle$.

### 1.4 Singularity Localization (1D vs nD)

*   **1D / Separable Reductions:** Finite-time blow-up $T(x_0)$ is explicitly evaluated via exact quadrature: $T(x_0) = \int_{x_0}^\infty \frac{dx}{f(x)}$.
*   **nD Coupled Flow:** Singularity is localized numerically by integrating the classical RHS $f_i(x)$ with an adaptive explicit solver (e.g., DOP853). Blow-up is triggered when $\|x\| \to \infty$ or when the solver's adaptive step $\Delta t \to 0$ due to spatial gradients.
